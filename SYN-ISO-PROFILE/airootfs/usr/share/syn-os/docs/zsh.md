# The shell

SYN-OS uses zsh, with a couple of quality-of-life touches set up for you
already: shared history across sessions, a prompt that shows your
username, folder, git branch, and the time, and a green or red checkmark
so you can tell at a glance whether your last command succeeded.

A few handy shortcuts are already set up:

| Type this | To get |
|---|---|
| `synos` | Start the desktop |
| `ll` / `la` / `l` | Different flavors of `ls` |
| `cat`, `grep` | Nicer, faster replacements, if you have them installed |

Tab completion, useful history search, and sensible keybindings for
things like word-deletion are all set up from the start too.

## The live USB shell

Before you've installed anything, the shell you land in on the live USB
is a lighter setup just for getting the install done: it checks you're
online, reminds you to review your settings, and gives you the one
command you need to kick off the install.
