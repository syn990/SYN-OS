# What's a window manager?

A **desktop environment**, like GNOME or KDE, is a whole bundled suite:
window management, a panel, a file manager, notifications, all built and
themed together as one package by one team.

SYN-OS isn't one of those. Instead, it's made of separate, independently
built pieces, each chosen deliberately and wired together: LabWC handles
your windows, Waybar is the bar across the top, and SYN-OS's own tools
fill in the rest, like the file manager and notifications.

The **window manager** is just the piece deciding how your windows are
arranged, moved, resized, and focused, LabWC's job on SYN-OS.

Nothing here calls itself a "desktop environment," and that's the point.
Each piece stays independently swappable, and you can see and understand
exactly which piece does what.
