# Screenshots and screen recording

Two quick tools for grabbing what's on your screen: a screenshot tool and
a screen recorder. Both ask where to save before doing anything, and both
let you know when they're done.

## Screenshots

| Key | Does |
|---|---|
| `Super+p` | Screenshot a selected area |

Or use the Capture menu for a full-screen shot instead. Either way you'll
be asked where to save it (Pictures is the default), and it lands there
as a PNG named with the date and time.

## Screen recording

| Key | Does |
|---|---|
| `Super+Shift+p` | Start recording the full screen, press it again to stop |

The Capture menu also has an option to record just a selected area. Once
a recording is running, a "Recording" indicator shows up on the bar,
click it any time to stop.

If you've got more than one monitor, you'll be asked which one to record
when you start a full-screen capture.

Recordings save as `.mp4` files, named with the date and time, in the
folder you choose (Videos is the default).
