# Directory maps

Turns any folder into a visual map, a diagram showing every subfolder and
file branching out from it, colored to match your current theme. Handy
for getting a feel for a big project you don't know yet, or for showing
someone the shape of your files.

Find it under SYN-OS Tools in the main menu, with three options:

- **Quick** scans a couple of levels deep. Good for a fast overview.
- **Full** scans everything.
- **Custom** lets you pick exactly how deep to go.

Pick a folder, pick PNG or SVG for the output, and it draws the map for
you. The finished image lands in `~/Pictures/SYN-GRAPHMAP/`.
