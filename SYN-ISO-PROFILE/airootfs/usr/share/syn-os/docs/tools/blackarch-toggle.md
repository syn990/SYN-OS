# BlackArch tools

[BlackArch](https://blackarch.org) is a huge collection of security and
penetration-testing tools. SYN-OS doesn't install it by default, but you
can turn it on any time from Preferences → BlackArch, no reinstall
needed.

Turning it on adds BlackArch's package repository, updates your system,
and installs a small, curated set of tools rather than the entire
collection, which would be thousands of packages. Once enabled, you'll
find a BlackArch section in the Applications menu.

Turning it off removes everything it added, cleanly, back to how things
were before.

You'll see a notification once it's finished either way. After enabling
or disabling, press `Super+Escape` to refresh the menu so the change
shows up.
