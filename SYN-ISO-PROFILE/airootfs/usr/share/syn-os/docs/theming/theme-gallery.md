# Theme gallery

All 63 themes SYN-OS ships, grouped the same way you'll find them in
Preferences → Themes. Screenshots are on the way, for now this is a list
to browse by name.

## Dark

**Vanilla** (the original SYN-OS look): RED (the default), BLUE, GREEN,
M141, ORANGE, PINK, PURPLE, YELLOW, GRAPHITE, LIGHT

**Flatline** (the most minimal): MATRIX (classic green-on-black terminal
look), FLATLINE-CYAN, FLATLINE-MAGENTA, FLATLINE-LIME, FLATLINE-GOLD,
FLATLINE-ICE, FLATLINE-CRIMSON, FLATLINE-TEAL, FLATLINE-SLATE

**Slab** (thick borders, chunky blocks): SLAB-AMBER, SLAB-CRIMSON,
SLAB-FOREST, SLAB-COBALT, SLAB-VIOLET, SLAB-BRASS, SLAB-ROSE, SLAB-MONO

**Halo** (glowing outlines): HALO-VIOLET, HALO-CYAN, HALO-AMBER,
HALO-CRIMSON, HALO-LIME, HALO-ROSE, HALO-ICE, HALO-GOLD

**Bevel** (gradients and depth): BEVEL-STEEL, BEVEL-COPPER,
BEVEL-EMERALD, BEVEL-SLATE, BEVEL-WINE, BEVEL-BRONZE, BEVEL-TEAL,
BEVEL-PLUM

## Light

**Vanilla**: BRIGHT, SILVER, VANILLA-LIGHT-CREAM, VANILLA-LIGHT-FROST

**Flatline**: FLATLINE-LIGHT-INK, FLATLINE-LIGHT-ROSE,
FLATLINE-LIGHT-AMBER, FLATLINE-LIGHT-SKY

**Slab**: SLAB-LIGHT-STONE, SLAB-LIGHT-SAGE, SLAB-LIGHT-DUSK,
SLAB-LIGHT-CLAY

**Halo**: HALO-LIGHT-AZURE, HALO-LIGHT-CORAL, HALO-LIGHT-MINT,
HALO-LIGHT-ORCHID

**Bevel**: WIN95 (a nod to classic Windows 95), BEVEL-LIGHT-PEARL,
BEVEL-LIGHT-BLUSH, BEVEL-LIGHT-SEAFOAM

See [The theme system](./theme-engine.md) for how switching themes
actually works.
